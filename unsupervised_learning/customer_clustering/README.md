# Customer Clustering (Online Retail II)

This project explores customer segmentation using the Online Retail II dataset and K‑Means clustering.

## Project structure
- `unsupervised_project/clustering/` — main clustering project
  - `data/` — dataset file
  - `notebooks/` — analysis notebooks
  - `images/` — plots and figures

## How to run
1. Download the dataset from the official source and place it in:
   `unsupervised_project/clustering/data/online_retail_II 4.xlsx`
2. Open the notebook:
   `unsupervised_project/clustering/notebooks/k_means_customer.ipynb`
3. Run all cells from top to bottom.

## Dataset
Official source (UCI Online Retail II):
https://archive.ics.uci.edu/dataset/502/online+retail+ii
